Quick Start

Get up and running in minutes

# Clone the repository
git clone https://github.com/Timkey/pptx-Slide-manager.git
cd pptx-Slide-manager

# Install dependencies
pip install -r src/requirements.txt

# Merge presentations with JSON config
python scripts/python/merge_pptx.py --config assets/config/merge-config.json

# Analyze font attributes
python scripts/python/analyze_pptx_fonts.py /path/to/presentations --output assets/analysis/output

# Generate interactive visualization
python scripts/python/generate_analysis_viz.py assets/analysis/output

Main Scripts

merge_pptx.py

Production: Merge multiple PPTX files with theme resolution, layout matching, and JSON configuration support.

analyze_pptx_fonts.py

Analysis: Extract font attributes with deep theme resolution and outlier detection across presentations.

generate_analysis_viz.py

Visualization: Create interactive HTML reports with charts, search, and color swatches - no server needed.

validate_merge.py

Validation: Comprehensive validation of merged outputs checking slides, shapes, text, and formatting.
